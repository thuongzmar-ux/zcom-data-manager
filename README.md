# ZCOM Data Manager

Dashboard quản lý data khách hàng, pipeline bán hàng và hoa hồng cho ZCOM Agency.

## Tính năng
- 📋 **Pipeline**: Quản lý toàn bộ data khách (HOT/WARM/COLD), phân công người phụ trách, stage tracking
- 📊 **Doanh số**: Theo dõi revenue theo 4 nhóm sản phẩm (ZME/ZMKT/ZHUB/Mercy Tech) và GAP vs định phí
- 💰 **Lương B**: Tự tính hoa hồng (20%) theo cơ chế originate/close

## Mô hình tài chính
- CP SX/VH: 30% | Hoa hồng: 20% | Marketing: 5% | Contribution: 45%
- Breakeven: **340,000,000₫/tháng**
- Mercy Tech: 60tr | ZCOM: 280tr

## Cách dùng
Mở file `index.html` trong trình duyệt. Data lưu vào localStorage — không cần server.
